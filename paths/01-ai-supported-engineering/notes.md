# Path 1: notes

## Why the path looks the way it does

The 30-minute tier holds the vendor primary plus one independent counterweight. The 3-hour tier holds the canonical reference for the wider AIDLC framing plus the engineering-org-design lens. The 30-hour tier asks for a shipped artefact, not a finished book.

The path deliberately omits vendor-comparison content. Comparing Claude Code to Cursor on a feature matrix is exhausting, and the comparison changes every quarter. The substrate-shape framing in `AIDLC.md` §3 lasts longer.

## Resources considered but not included

- Multiple vendor blog posts on agentic coding workflows. Excluded for the vendor-tilt drift the wider repo avoids.
- A general-purpose book on AI-assisted development. No current candidate passes the operator-tested bar at the firm sizes the path is written for. When one ships that does, it will replace either the Larson entry or the harness exercise.
- "Watch one talk" entries. Conference-talk half-life is too short; holding off.
